## Sistema de Crud Simples 

Sistema de Crud simples, Pessoas
Tecnologias => PHP 7.4/Laravel 7.25 - Bootstrap 4/Jquery

## Laravel
Inicialize o projeto
```sh
composer install
```
Rode as migrations
```sh
php artisan migrate
```
Rode as Seeders
```sh
php artisan db:seed
```
